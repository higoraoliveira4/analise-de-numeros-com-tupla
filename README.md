# Análise de Números com Tupla (Python)

Projeto de estudo desenvolvido em Python com foco no uso de tuplas para armazenar dados e realizar operações básicas de análise, como contagem, busca e verificação de valores.

---

## Objetivo do projeto

Praticar conceitos fundamentais da linguagem Python, como:

  * Tuplas
  * Entrada de dados pelo usuário (`input`)
  * Métodos de tupla (`count()` e `index()`)
  * Estruturas condicionais (`if/else`)
  * Estruturas de repetição (`for`)
  * Operadores aritméticos (`%` para verificação de números pares)
    
* Desenvolver lógica de programação
* Trabalhar com validação e análise de dados simples

---

## Funcionalidades

* Recebe quatro números inteiros informados pelo usuário
* Armazena os valores em uma tupla
* Conta quantas vezes o número 9 aparece
* Verifica se o número 3 foi digitado e informa sua posição
* Identifica quantos valores são números pares

---

## Tecnologias utilizadas

* Python 3
* Execução via terminal ou IDE (PyCharm, VS Code, etc.)

Como executar:

- Certifique-se de que o Python 3 está instalado no computador
- Baixe o arquivo deste repositório
- Execute o arquivo pelo terminal ou por uma IDE de sua preferência

---

## Exemplo de uso

- Digite um numero: 2
- Digite outro numero: 9
- Digite mais um numero: 3
- Digite o ultimo numero: 4

Saída:

- O número 9 apareceu 1 vez na lista.
- O número 3 apareceu na posição 3
- 2 valores são número par.
